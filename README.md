# avr-fat-cpp

Lightweight FAT filesystem for AVR microcontrollers using SDCard I/O.
Based on [greiman](https://github.com/greiman)'s [SdFat](https://github.com/greiman/SdFat) library.

## Built with

* avr-g++ (GCC) 8.2.0
